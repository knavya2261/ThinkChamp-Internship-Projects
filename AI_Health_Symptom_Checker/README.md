# AI Health Symptom Checker

## Project Description

This project is developed using Python.

It accepts symptoms from the user and predicts a possible disease.

The project also performs data analysis, generates charts, and creates a health report.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Files

- app.py
- symptom_checker.py
- data_analysis.py
- visualizations.py
- report_generator.py
